:: HOW TO // DEUTSCH

   [1] "Install.bat" (per Rechtsklick > Als Admin ausführen) installiert alle Miner auf C:\Apps\Crypto\Miners.
   [2] Konfig (Wallets,...) in C:\Apps\Crypto\Miners\_config\setup.bat anpasssen. (Rechtsklick>Bearbeiten!)
   [3] Beliebigen Miner per Klick auf das Desktop-Symbol starten und einfach Coin auswählen!

   [!] Es kann sein das ein Virusmeldung kommt. Diese kommt auch bei den Original-App-Downloads. :/
   [!] Manche Miner ***MÜSSEN*** einmal als Administrator direkt gestartet werden (siehe Claymore Readme)!

:: HOW TO // ENGLiSH

   [1] Run "Install.bat" (rightclick > Run as Admin) to install all Miners to C:\Apps\Crypto\Miners.
   [2] Adjust config (Wallets,...) in C:\Apps\Crypto\Miners\_config\setup.bat (Rightclick>Edit!)
   [3] Launch whatever Miner via Desktop shortcut and select your Coin of choice!

   [!] Your Antivirus might show a false positive on Claymore Miners. Nothing i can do about :/
   [!] Some Miners ***NEED*** to be launched as Admin initially (Check Claymore Readme for more)!
